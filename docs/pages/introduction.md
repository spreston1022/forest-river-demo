# Introduction

Welcome to the **Forest River API** developer portal. This portal gives you everything you need to integrate with the Forest River platform — from browsing the API reference to provisioning your own API keys.

## What is the Forest River API?

The Forest River API is a RESTful HTTP API that allows developers to build applications and automations on top of the Forest River platform. Whether you're pulling data, triggering workflows, or building a full integration, the API gives you programmatic access to the resources you need.

## Key features

- **Simple authentication** — Secure every request with an API key passed in the `Authorization` header
- **Consistent responses** — All endpoints follow the same JSON response structure with predictable error codes
- **High availability** — Backed by an enterprise-grade API gateway with global edge delivery
